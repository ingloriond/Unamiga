## Teclas:

#### F1  :    1P Start
#### F2  :    2P Start
#### F3  :    Insert Coin
#### F4: Scanlines (alternando entre no scanlines, 25%, 50% y 75%)
#### ESQ :    Reset
#### BLOQ DESP: Cambio entre VGA y Rgb)
#### Controles: Joystick y Teclado (Cursores y espacio el salto)
